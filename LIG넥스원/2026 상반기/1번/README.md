# 문제 1. 암호화된 잔고 복원

## 문제 설명

암호화된 잔고 문자열 `balance`가 주어집니다.
문자열은 숫자(`0`~`9`)와 대문자 알파벳 `A`, `B`로만 이루어져 있습니다.

문자열을 왼쪽부터 차례대로 읽으며 실제 잔고 문자열을 복원하려고 합니다.
복원 규칙은 다음과 같습니다.

- 숫자 문자는 그대로 결과 문자열의 뒤에 붙입니다.
- `A`는 **지금까지 복원된 문자열에 포함된 모든 숫자의 합**을 10진수 문자열로 변환해 결과 문자열의 뒤에 붙입니다.
- `B`는 **지금까지 복원된 문자열의 마지막 두 자리 숫자의 차의 절댓값**을 10진수 문자열로 변환해 결과 문자열의 뒤에 붙입니다.

항상 입력은 유효하게만 주어집니다.

복원이 완료된 최종 문자열을 return 하도록 `solution` 함수를 완성해 주세요.

---

## 제한사항

- `1 <= len(balance) <= 100`
- `balance`는 숫자와 대문자 `A`, `B`로만 이루어져 있습니다.
- `balance[0]`은 항상 숫자입니다.
- `balance[1]`이 존재한다면 숫자 또는 `A`입니다.

---

## 입출력 예

| balance  | result     |
| -------- | ---------- |
| `"55AA"` | `"551011"` |
| `"55AB"` | `"55101"`  |
| `"7A"`   | `"77"`     |
| `"123B"` | `"1231"`   |

---

## 입출력 예 설명

### 예시 1

- `5` -> `"5"`
- `5` -> `"55"`
- `A` -> 현재 숫자 합은 `5 + 5 = 10`, 따라서 `"5510"`
- `A` -> 현재 숫자 합은 `5 + 5 + 1 + 0 = 11`, 따라서 `"551011"`

### 예시 2

- `5` -> `"5"`
- `5` -> `"55"`
- `A` -> `"5510"`
- `B` -> 마지막 두 자리 숫자는 `1`, `0`이고 차의 절댓값은 `1`, 따라서 `"55101"`

### 예시 3

- `7` -> `"7"`
- `A` -> 현재 숫자 합은 `7`, 따라서 `"77"`

### 예시 4

- `1`, `2`, `3`을 그대로 붙이면 `"123"`
- `B` -> 마지막 두 자리 숫자는 `2`, `3`이고 차의 절댓값은 `1`, 따라서 `"1231"`
