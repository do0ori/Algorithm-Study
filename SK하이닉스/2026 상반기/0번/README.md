# 사전 테스트 문제. 정규표현식 축약

## 문제 설명

정규표현식(Regular Expression, regex)은 특정한 규칙을 가진 문자열의 집합을 표현하는 형식 언어입니다.

정규표현식에 있는 패턴들 중 `*` 는 **빈 문자열**, 또는 **같은 문자가 1번 이상 반복되는 패턴**을 의미하며, 다음과 같은 예시들이 있습니다.

- `a*` 는 `a` 가 0개 이상 나열된 문자열의 집합을 표현합니다.  
  이 정규표현식으로 표현할 수 있는 문자열들은 `""`(빈 문자열), `"a"`, `"aa"`, `"aaa"` 등이 있습니다.

- `ab*ccb*` 는 `"a" + "b*" + "cc" + "b*"` 를 의미합니다.  
  이 정규표현식은 `a` 가 1개 먼저 나타나고, 그다음에 `b` 가 0개 이상, 그다음에 `c` 가 2개, 마지막으로 다시 `b` 가 0개 이상 나타나는 모든 문자열의 집합을 표현합니다.  
  이 정규표현식으로 표현할 수 있는 문자열들은 `"acc"`, `"abcc"`, `"abbbccbbb"` 등이 있습니다.

알파벳 소문자로 이루어진 문자열 `line` 이 매개변수로 주어집니다.  
`line` 을 표현하는 **알파벳과 별(`*`)로 이루어진 가장 짧은 정규표현식**을 return 하도록 `solution` 함수를 완성해주세요.

단, 가장 짧은 정규표현식이 여러 개라면, **별(`*`)이 가장 많이 포함된 문자열**을 return 해야 합니다.

---

## 제한 사항

- `line`의 길이는 `1` 이상 `10,000` 이하입니다.
- `line`은 `'a'`부터 `'z'`까지의 소문자로만 이루어진 문자열입니다.

---

## 입출력 예

| line           | result        |
| -------------- | ------------- |
| `"aabbbc"`     | `"a*b*c"`     |
| `"helllllllo"` | `"hel*o"`     |
| `"wonderful"`  | `"wonderful"` |

---

## 입출력 예 설명

### 예시 1

`"aabbbc"` 를 표현하는 가장 짧은 정규표현식은 `"aab*c"` 와 `"a*b*c"` 입니다.  
이 중에서 별(`*`)이 더 많이 포함된 정규표현식인 `"a*b*c"` 를 return 해야 합니다.

### 예시 2

`"helllllllo"` 를 표현할 수 있는 가장 짧은 정규표현식은 `"hel*o"` 입니다.

### 예시 3

`"wonderful"` 은 그 자체로 `"wonderful"` 을 표현할 수 있는 가장 짧은 정규표현식이며,  
주어진 조건을 만족하면서 이보다 더 짧은 정규표현식을 만드는 것은 불가능합니다.
