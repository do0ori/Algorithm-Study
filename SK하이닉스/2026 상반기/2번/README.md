# 문제 2. 연속하는 자연수 두 개 이상을 곱해 만들 수 있는 수의 n번째 원소

## 문제 설명

연속하는 자연수 두 개 이상을 곱해서 만들 수 있는 자연수를 작은 수부터 나열한 수열이 있습니다.

예를 들면 다음과 같습니다.

```text
2, 6, 12, 20, 24, 30, 42, 56, 60, 72, ...
```

이 수열의 `n`번째 수를 반환하세요.

## 제한사항

- `1 ≤ n ≤ 1,000,000`
- 정답은 `10^12` 이하

## 풀이 개요

어떤 수가 수열에 포함되려면,

```text
a * (a+1) * ... * (a+k)
```

형태여야 하며, 길이는 최소 2입니다.

정답이 `10^12` 이하라고 했으므로, `10^12` 이하의 모든 연속곱을 생성한 뒤

- 정렬
- 중복 제거

를 하면 수열 전체를 얻을 수 있습니다.

## 시간복잡도

생성되는 연속곱 개수가 제한되어 있어 브루트포스로 충분합니다.
