# 문제 3. 규칙에 따른 배열 섞기

## 문제 설명

`1`부터 `n`까지 들어있는 배열에 대해 다음 연산을 반복합니다.

현재 배열 길이를 `len`이라 할 때,

1. `len`을 나누었을 때 나누어 떨어지는 가장 작은 소수 `p`를 구합니다.
2. 현재 배열을 `len / p` 길이인 `p`개의 작은 배열로 쪼갭니다.
3. 기존 배열의 값들을 순서대로 보면서, 쪼갠 배열들에 하나씩 차례대로 담습니다.
4. 각 작은 배열에 대해 같은 규칙을 재귀적으로 반복합니다.
5. 길이가 `1`이 되면 종료합니다.

예시:

```text
[1, 2, 3, 4, 5, 6, 7, 8]
-> (p = 2)
[1, 3, 5, 7], [2, 4, 6, 8]
-> (p = 2)
[1, 5], [3, 7], [2, 6], [4, 8]
-> (p = 2)
[1], [5], [3], [7], [2], [6], [4], [8]
=> [1, 5, 3, 7, 2, 6, 4, 8]
```

## 제한사항

- `2 ≤ n ≤ 1,000,000`

## 풀이 개요

현재 배열 길이의 최소 소인수 `p`를 구한 뒤,
`p`개의 버킷으로 round-robin 분배하고 각 버킷을 재귀 처리합니다.

최소 소인수는 `spf` 배열(최소 소인수 테이블)을 만들어 빠르게 구합니다.
